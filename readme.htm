c862hspについて

c862hspの概要、使い方、ライセンスなどについて解説します。

c862hspの概要

c862hspとは、HSP3のスクリプトとして、C++のソースコードをもとに、 オブジェクトファイルから機械語を抽出して出力するWindowsアプリケーションです。

c862hspの特徴としては以下のようなものがあります。

  • 機械語で処理の高速化を図ることができる
  • 関数を使うことができる
  • グローバル変数や文字列を使うことができる
  • 複数の関数定義ができる
  • DLLを使うことができる
  • 複数のオブジェクトファイルをもとにできる
  • スタティックライブラリからも変換できる

ソフトの使い方

C/C++のソースを変換する場合にはVisualC++を先にインストールしてください。 リンク時のコード生成を有効にしていると変換できません。

付属のsampleproj2は、設定の参考どして同梱しています。変換はできません。

ソフトを起動するには

インストールしたフォルダのc862hsp.exeを開きます。

プロジェクトを新しく作るには

  1. メイン画面で、プロジェクトボタンを押します
  2. 次に現れた画面で、新規ボタンを押します
  3. メッセージが現れるので、読んで、ＯＫを押します
  4. プロジェクトを保存するファイルを指定します
  5. メッセージが現れるので、読んで、ＯＫを押します
  6. 出力先hspファイルを、拡張子を含めて指定します
  7. 完了です

プロジェクトにソースファイルを追加するには

  1. プロジェクトを用意します
  2. メイン画面で、ファイルボタンを押します
  3. 次に現れた画面で、 新規ボタンを押します
  4. また次に現れた画面で、ソースコードボタンを押します
  5. 新しいソースファイルに、拡張子も含めて名前を付けます
  6. 続行するか聞かれるので、はいと答えます
  7. そのソースファイルによって生成される予定のオブジェクトファイル名を（フルパスで）指定します
  8. そのソースファイルをコンパイルするためのコマンドラインを指定します（ただし、バッチファイルを扱うときはcall hoge.batのようにします）
  9. 決定ボタンを押します
  10. 完了です
  11. メイン画面、上のエディタで編集し、上書きボタンで保存できます

オブジェクトやライブラリを追加するには

既存のライブラリを追加したい場合もあると思います。 また、DLL・APIを使うときは、対応したインポートライブラリを指定する必要があります。

  1. メイン画面で、決定ボタンを押します
  2. 次に現れた画面で、新規ボタンを押します
  3. また次に現れた画面で、objまたはlibボタンを押します
  4. 追加したいファイルを指定します
  5. 完了です

変換するには

  1. メイン画面で、変換ボタンを押します
  2. しばらく待ちます
  3. 完了です
  4. 出力表示ボタンで結果が見られます

出力されたファイルの使い方

  1. 出力されたファイルのinclude 
      #include "出力名.hsp"
  2. 関数ポインタの取得 
      funcptr = getsymptr_変換ＩＤ("シンボル名")
  3. 実行 
      prm = p1,p2,pn
  res = callfunc(prm,funcptr,n)
